    Algorithm Analysis

    Reference time and space complexity alongside practical trade-offs.

    Algorithm Time Complexity Space Complexity Advantages Disadvantages
    Brute Force O(n*m)
    n = text length, m = pattern length    		 O(1) Simple implementation, no preprocessing required Less efficient for large texts, performs redundant comparisons
    Rabin-Karp O(n+m) avg
    O(n*m) worst case    		 O(1) Good average-case performance, efficient for multiple patterns Hash collisions can degrade performance, worst-case is slow
    Knuth-Morris-Pratt (KMP) O(n+m) O(m) Guaranteed linear time, no backtracking in text Requires preprocessing (LPS array), additional space
